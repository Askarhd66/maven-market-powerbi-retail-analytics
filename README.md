<h1>🛒 Maven Market Retail Performance Dashboard</h1>

<h2>📌 Overview</h2>
<p>
This project presents an end-to-end Power BI analytics solution for <b>Maven Market</b>, a multi-national 
grocery chain operating across the <b>United States, Canada, and Mexico</b>. The dashboard provides a unified 
view of transactions, profitability, returns, and regional performance, enabling data-driven decision-making 
for retail operations and executive leadership.
</p>

<h2>📊 Key Monthly Metrics</h2>
<ul>
  <li><b>18,325</b> Monthly Transactions (Goal: 17,339, <b>+5.69%</b>)</li>
  <li><b>$71,682</b> Monthly Profit (Goal: $67.87K, <b>+5.61%</b>)</li>
  <li><b>496</b> Monthly Returns (Goal: 482, <b>-2.9%</b>)</li>
</ul>

<h2>🔧 What I Implemented</h2>
<ul>
  <li>Designed a complete <b>star schema</b> integrating customers, products, stores, regions, and returns</li>
  <li>Performed data cleaning and transformation using <b>Power Query</b></li>
  <li>Developed <b>30+ DAX measures</b> including KPIs, YTD/MTD metrics, profitability, and return rates</li>
  <li>Built interactive dashboards with <b>drill-through</b> and <b>dynamic filtering</b></li>
  <li>Created country-level comparisons for USA, Mexico, and Canada</li>
</ul>

<h2>📈 Dashboard Features</h2>
<ul>
  <li><b>Executive KPI Summary</b> for quick performance insights</li>
  <li><b>Geographical Performance</b> across North America</li>
  <li><b>Brand-Level Profitability</b> and return analysis</li>
  <li><b>Weekly Revenue Trending</b> for operational planning</li>
  <li><b>Revenue vs Target</b> gauge for performance tracking</li>
</ul>

<h2>📂 Datasets Used</h2>
<ul>
  <li><b>Customers</b> – demographics, account details, household attributes</li>
  <li><b>Products</b> – brand, pricing, SKU, cost, weight</li>
  <li><b>Stores</b> – location, size, remodel history, region mapping</li>
  <li><b>Regions</b> – sales districts and regions</li>
  <li><b>Calendar</b> – date intelligence table</li>
  <li><b>Transaction_Data</b> – revenue, profit, quantity, targets, YTD metrics</li>
  <li><b>Return_Data</b> – return quantities, rates, and historical return metrics</li>
